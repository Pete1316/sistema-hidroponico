# 🚀 GUÍA RÁPIDA - AUTOMATIZACIÓN DE NUTRIENTES

## ¡Haz esto AHORA! (en 3 pasos)

### 1️⃣ Abre la interfaz
```
Pestaña: "Diagnóstico del Cultivo"
```

### 2️⃣ Completa 3 campos
```
✏️ Días: (cuántos días desde que sembraste)
✏️ Etapa: (Semilla / Vegetativa / Floración / Producción)
✏️ Volumen: (litros de tu tanque)
```

### 3️⃣ Clic en botón rojo
```
Botón: ⚙️ Automatizar Nutrientes
```

**¡Listo!** Te dirá exactamente qué hacer.

## ✅ Si el sensor marca...

### **350 ppm** (bajo para Vegetativa)
→ Sistema dice: "Agrega 2.5g de N, 0.67g de P..."

### **600 ppm** (perfecto para Vegetativa)
→ Sistema dice: "Todo bien, sigue monitoreando"

### **1100 ppm** (alto para Vegetativa)
→ Sistema dice: "Cambia 45 litros de agua"

## ⚠️ Lo Más Importante

```
✓ Agregar nutrientes lentamente (no >200 ppm por dosis)
✓ Esperar 10 minutos entre cada nutriente
✓ Cambios de agua en máximo 50% de una vez
✓ Siempre medir después de 5 minutos
✓ Si hay mucha diferencia, repetir en 2-3 horas
```
